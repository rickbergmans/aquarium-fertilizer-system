# Aquarium fertilizer system
Arduino based aquarium fertilizer system, using a peristaltic pump.

**Components used:**
- Arduino Uno
- 12V peristaltic pump 
- 12V power source
- Transistor PN2222A NPN 40V / 1A
- Diode 1N4001 50V / 1A
- RTC DS3231 _(required for accurate time)_
